### Теоретические основы

#### Что такое Performance Testing?

Performance Testing - это процесс определения скорости, отзывчивости и стабильности компьютерной системы под рабочей нагрузкой.

│ │                                                            │   │

│ │   it('should get user details', async () => {             │   │
│ │     await provider                                         │   │
│ │       .given('user exists')                               │   │
│ │       .uponReceiving('a request for user details')        │   │
│ │       .withRequest({                                       │   │
│ │         method: 'GET',                                     │   │
│ │         path: '/users/123',                                │   │
