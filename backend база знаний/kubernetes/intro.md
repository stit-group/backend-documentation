## 🤔 Что такое Kubernetes и зачем он нужен?

**Kubernetes** (часто сокращается как **K8s**) — это платформа для автоматизации развертывания, масштабирования и управления контейнеризованными приложениями. Проще говоря, это "операционная система для облака", которая берет на себя всю сложность управления множеством контейнеров.

### Эволюция развертывания приложений:

```
Физические серверы → Виртуальные машины → Контейнеры → Kubernetes
     (2000s)              (2010s)           (2015+)        (2017+)
```

│  💰 БИЗНЕС-ЦЕЛИ:                                            │
│  ├─ Снижение операционных расходов                          │

│                                                             │
