  - Sliding window protocol
  - TCP состояния и их влияние на производительность

  - Socket options (SO_REUSEADDR, SO_KEEPALIVE)

  - Настройка буферов (SO_SNDBUF, SO_RCVBUF)
  - Connection pooling considerations

### Неделя 4: UDP и альтернативные протоколы
- **UDP протокол**
  - Отличия от TCP

  - UDP для real-time приложений

- **QUIC протокол**
  - Принципы работы

  - Реализация в backend сервисах

6. Настройка TLS для production окружения
